---
title: "Settings"
---

:::note
Find these under your TradingCards/settings folder
:::

Use these files to control plugin-wide behavior before you start editing card content:

- [General](./general.md): default materials, display formats, deck behavior, plugin support, and misc options
- [Chances](./chances.md): drop odds and rarity weighting
- [Messages](./messages.md): user-facing text and prefixes
- [Storage](./storage.md): YAML or MySQL storage configuration
- [Advanced](./advanced.md): cache tuning and low-level performance options

Most servers only need `general.yml`, `chances.yml`, and `messages.yml` at first. Leave the advanced cache settings alone unless you have a specific performance problem to solve.
