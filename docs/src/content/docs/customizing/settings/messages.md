title: "Messages"
---

:::note
The default file is in the plugin jar at `settings/messages.yml`, and the source copy is under `tradingcards-plugin/src/main/resources/settings/messages.yml`.
:::

Use `messages.yml` to customize player-facing text such as:

- Command feedback
- Economy and reward messages
- Giveaway broadcasts
- Error responses
- Inventory and section titles

## Placeholders

Different messages support different placeholders. Common ones include:

- `%player%`
- `%card%`
- `%pack%`
- `%amount%`
- `%rarity%`
- `%uuid%`

Keep your server formatting codes consistent with the rest of the plugin so cards, decks, and chat output feel aligned.
