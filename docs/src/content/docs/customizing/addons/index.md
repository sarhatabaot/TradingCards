TradingCards supports a core plugin plus optional integrations and companion plugins.

Use this section when you need features that are not built into the base plugin or when you want to hook TradingCards into other parts of your server.

## In This Section

- [Official Addons](./official-addons.md)
- [Community Addons](./community-addons.md)

## Notes

- Some integrations are soft dependencies and simply enhance behavior when the other plugin is installed.
- Others are separate addon jars that must also be placed in the server's `plugins/` folder.
