This migration keeps YAML storage, but changes where several data files live.

## Steps

1. Back up your existing plugin data.
2. Move `custom-types.yml`, `rarities.yml`, `series.yml`, and `packs.yml` into the plugin `data` folder.
2. Restart your server.

## Verify After Restart

- Rarities load correctly
- Series appear in commands and menus
- Packs still resolve their configured cards
- Custom types still participate in drops
